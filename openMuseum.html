Open museum

Présentation du concept

Open museum est une application web permettant de visualiser les musées de la région PACA.
L'application se base sur une architecture REST et a été développée via la stack MEAN : MongoDB (Atlas), Express, Angular et Node.

Objectifs

  • Les musées doivent posséder une fiche d'informations avec tous les détails du musée
  • les musées doivent posséder une carte indiquant l'adresse du musée via le module leaflet
